Product Review Analysis

The project's analysis of product review data highlights actionable insights crucial for enhancing customer satisfaction and business ratings. Through a systematic approach involving data exploration, question formulation, and advanced data preparation using a Hugging Face pretrained model for zero-shot learning, the project categorized customer feedback into key aspects like comfortability and fitting.

July, 2023

Meta's Stock Price Prediction using ARIMA

In this project, I utilized ARIMA to construct a predictive model for forecasting Meta's stock price. The ARIMA model effectively captured short-term trends and variations, making it suitable for predicting Meta's stock price over approximately 30 days with reasonable accuracy.

The project features a step-by-step implementation of ARIMA, starting with exploratory data analysis (EDA) to uncover underlying patterns. Parameter estimation for the ARIMA model was guided by Autocorrelation Function (ACF) and Partial Autocorrelation Function (PACF) plots, ensuring the selection of appropriate model orders.

Throughout the project, diagnostic checks were performed to validate the ARIMA model's accuracy. These checks included visual inspection of residuals and model performance against actual stock prices. The robustness of the model was further assessed through residual analysis and comparison with historical data trends. While the ARIMA model demonstrated strong predictive performance for short-term forecasting, its application for longer-term predictions was cautioned due to the inherent volatility of financial markets.

May, 2023

Click Through Rate Prediction

In this project, we performed the exploratory data analysis of 14 daysâ€™ worth of mobile ad clicks dataset. We developed, trained, and evaluated models using Naive Bayes, XG Boost, Random Forest, Gradient Boost, and Logistic Regression. The main feature of this project is to show how to transform the categorical variables that have many values to make one-hot encoding an inappropriate method of transformation. Another highlight of this project is how to determine the optimum threshold for classification and present the trade-offs between recall and precession.

We did not create a balanced dataset. Instead, train and test on an imbalanced original dataset to reflect reality. Based on the evaluation, XG Boost and Gradient Boost have much better AUC than that of other models. However, we selected XG Boost on the grounds of resource requirement as Gradient Boost consumes a lot of resources.

We further evaluated the model using the cumulative gain chart and lift curve, which shows that selecting the top 20 % of the leads ranked as most likely to respond to an ad will contain about 55% of the actual click. The expected baseline number without using the model would only be 20%. Therefore, the model adds business value.
